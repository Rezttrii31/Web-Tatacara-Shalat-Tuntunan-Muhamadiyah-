FOLDER IMAGES
=============

Folder ini berisi gambar-gambar untuk ilustrasi gerakan sholat.

GAMBAR YANG DIPERLUKAN:
1. berdiri-tegak.png - Gambar posisi berdiri tegak dalam sholat
2. takbiratul-ihram.png - Gambar gerakan takbiratul ihram
3. rukuk.png - Gambar posisi rukuk
4. sujud.png - Gambar posisi sujud
5. duduk-antara-sujud.png - Gambar posisi duduk antara dua sujud
6. tahiyat-akhir.png - Gambar posisi duduk tahiyat akhir
7. salam.png - Gambar gerakan salam

SPESIFIKASI GAMBAR:
- Format: PNG (direkomendasikan untuk transparansi)
- Ukuran: Minimal 400x300 pixel
- Rasio: 4:3 atau 16:9
- Kualitas: Tinggi (minimal 72 DPI)
- Background: Transparan atau putih
- Style: Ilustrasi yang jelas dan mudah dipahami

CATATAN:
- Pastikan gambar memiliki hak cipta yang sesuai
- Gunakan gambar yang sopan dan sesuai dengan nilai Islam
- Gambar sebaiknya menunjukkan gerakan sholat yang benar
- Jika tidak ada gambar, ikon Font Awesome akan tetap digunakan sebagai fallback
- Sistem fallback otomatis: jika gambar tidak ditemukan, ikon akan ditampilkan
- Semua gambar akan responsive dan memiliki efek hover

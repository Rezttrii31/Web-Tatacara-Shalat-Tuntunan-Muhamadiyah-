DIREKTORI AUDIO
===============

Direktori ini berisi file-file audio untuk setiap tatacara sholat.

File yang diperlukan:
- berdiri.mp3     - Audio penjelasan posisi berdiri
- takbir.mp3      - Audio penjelasan takbiratul ihram
- rukuk.mp3       - Audio penjelasan gerakan rukuk
- sujud.mp3       - Audio penjelasan gerakan sujud
- duduk.mp3       - Audio penjelasan duduk antara dua sujud
- tahiyat.mp3     - Audio penjelasan tahiyat akhir
- salam.mp3       - Audio penjelasan gerakan salam

Format yang didukung:
- MP3 (direkomendasikan)
- WAV
- OGG

Spesifikasi audio yang direkomendasikan:
- Bitrate: 128 kbps atau lebih tinggi
- Sample Rate: 44.1 kHz
- Durasi: 30-60 detik per file
- Kualitas: Stereo atau Mono

Catatan:
- Jika file audio tidak tersedia, website akan menampilkan informasi teks sebagai gantinya
- Pastikan file audio memiliki kualitas yang baik dan jelas
- File audio sebaiknya berisi penjelasan dalam bahasa Indonesia
- Durasi audio tidak boleh terlalu panjang untuk menjaga user experience
